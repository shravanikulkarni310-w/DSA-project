# 📊 DSA Array Visualizer

An interactive **Array Visualizer** built using **HTML, CSS, and JavaScript** to help students understand fundamental array operations through smooth animations and a modern user interface.

---

## 🚀 Live Preview

> Clone or download the repository and open `index.html` in your browser.

---

## ✨ Features

- 📥 Create a custom array
- 🎲 Generate a random array
- ➕ Insert an element at any index
- ➖ Delete an element from any index
- 🔄 Reverse the array with animation
- 🗑️ Reset the array
- 📏 Display current array size
- ⏱️ Show time complexity for operations
- 🎚️ Adjustable animation speed
- 🌙 Dark mode support
- 📱 Fully responsive design
- 🎨 Modern brown-themed user interface

---

## 🛠️ Technologies Used

## 🎯 Supported Operations

| Operation | Time Complexity |
|-----------|-----------------|
| Create Array | O(n) |
| Insert at Index | O(n) |
| Delete at Index | O(n) |
| Reverse Array | O(n) |
| Random Array | O(n) |
| Reset Array | O(1) |
